Tentang Desain Komunikasi Visual

Dipublikasikan: 3 Februari 2026

Gambar ilustrasi DKV

Desain Komunikasi Visual (DKV) adalah cabang ilmu desain yang mempelajari konsep komunikasi dan ungkapan kreatif melalui berbagai media untuk menyampaikan pesan dan gagasan secara visual.

DKV mencakup berbagai bidang seperti desain grafis, ilustrasi, fotografi, multimedia, animasi, dan pengembangan web. Sebagai mahasiswa DKV, saya belajar menggabungkan elemen visual dengan teknologi untuk menciptakan karya yang menarik dan komunikatif.

Perjalanan Belajar HTML & CSS

Dipublikasikan: 27 Januari 2026

Ilustrasi Coding

Belajar HTML dan CSS adalah langkah pertama dalam memahami pengembangan web. HTML memberikan struktur pada website, sementara CSS menambahkan gaya dan keindahan visual.

Dalam perjalanan belajar ini, saya telah mempelajari berbagai konsep seperti layout CSS, background, font styling, dan responsiveness. Website ini adalah hasil dari pembelajaran tersebut - dibuat dengan HTML dan CSS murni tanpa framework.
